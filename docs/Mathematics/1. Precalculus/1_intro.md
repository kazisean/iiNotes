# Precalculus for Calculus  
#### Prerequisite : Knowledge of basic arithmetic.

This guide follows the book *Precalculus: Mathematics for Calculus* by [James Stewart.](https://en.wikipedia.org/wiki/James_Stewart_(mathematician)) This book is well respected & popularly recommended to help students prepare for their Calculus courses. University like NYU are using this book to teach their Precalculus classes. Please know that you do not need to buy the book to follow along with this guide. Although we highly recommend it. 


 ![Precalculus: Mathematics for Calculus Book Cover](https://i.ebayimg.com/images/g/RI4AAOSwULpf~uVa/s-l500.jpg)


The guides for each chapter would consist of YouTube videos for every sections. It would also include many different problem sets you can solve on your own for better practice. While you are doing the problems make sure you know that the purpose of doing them is in order to understand how they work. You want to figure out how a problem is pieced together. Once you understand the underlying principle you would never forget it again or get this type of math wrong. It is encouraged that you get the problems wrong when you first start to do them. The reason being it makes you question why the way you approached the problem wasn't correct. Through trial and error you figure out the correct approach. Problem solving in math is more like self discovering. Teaching yourself why something works and doesn't work. It is not fun if you don't get a few wrongs where and there. And expect exponential growth. You are not going to know everything when you first start a chapter. You have to give it time and learn each topics slowly. You are going to get many problems wrong before you start never getting them wrong again. I will leave this page with this quote : 

> "Any Fool Can Know. The Point Is To Understand" - Albert Einstein
