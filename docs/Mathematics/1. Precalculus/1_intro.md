# Precalculus for Calculus  
#### Prerequisite : Knowledge of basic arithmetic.

This guide follows the book *Precalculus: Mathematics for Calculus* by [James Stewart.](https://en.wikipedia.org/wiki/James_Stewart_(mathematician)) This book is well respected & popularly recommended to help students prepare for their Calculus courses. University like NYU are using this book to teach their Precalculus classes. Please know that you do not need to buy the book to follow along this guide. Although we highly recommend it.
