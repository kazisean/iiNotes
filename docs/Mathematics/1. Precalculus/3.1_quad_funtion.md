### Introduction :

A Quadratic Function is a type of function that is useful in many areas of mathematics, science and engineering. For example it is used to model motion, acceleration and force in Physics. It is also used during financial analysis, computer graphics and lot more. If you want to be a aerospace engineer or a computer programmer or even a financial analyst it is crucial to understand how Quadratic Functions work.

The quadratic formula is the following: 

$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

In this case a,b and c is a constant the value of x is unknown. You can plug in the value of a,b and c to solve for x.
Note: Quadratic Formula can only be used to solve Quadratic equations. If you have a polynomial equation for example you would not be able to use this formula.
